## What is This?

## How to Run the Tests

### 1) Update Java to at least 1.8.9_91
  
  [Click Here](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)

### 2) Install Gradle
  
  ``` brew install gradle ```

### 3) Run the Tests!

  ``` gradle test ```
